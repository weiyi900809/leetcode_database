
27. Remove Element

class Solution {
public:
    int removeElement(vector<int>& nums, int val) {
        int arr_size=nums.size();
        int read_ptr=0;
        int write_ptr=0;
        for(read_ptr=0; read_ptr < arr_size ;  read_ptr++){
            if(nums[read_ptr]!=val){
                nums[write_ptr]=nums[read_ptr];
                write_ptr++;
            }
            
        }
        return write_ptr;
    }
};
/*
破關提示（讀寫指標法）：
準備一個慢指標（寫入位置 slow = 0）。

用一個 for 迴圈當作快指標（讀取位置 i），無腦把陣列掃過一次。

核心邏輯： 只要快指標讀到的數字 「不是」那個討人厭的 val，就把它寫進慢指標的位置裡，然後慢指標往前走一步（slow++）。

如果讀到的是 val 怎麼辦？什麼都不要做，快指標直接跳過它。
*/
