leetcode 191 Number of 1 Bits

class Solution {
public:
    int hammingWeight(int n) {
        long int counter=0;
        while(n != 0){
            n = n & (n - 1); //將n的最右邊的1變成0
            counter++;
        }
        return counter;
    }
};

這題直接用之前學過的技巧來解決就行 



num & (num - 1)：直接把最右邊的 1 抹除掉（變成 0）。

我們一樣拿 12 (1100) 來看：
1100 & 1011，結果直接變成 1000。最右邊的那個 1 直接不見了！
所以你只要不斷執行 num = num & (num - 1)，每次抹除一個 1，計數器就加 1。
這個數字裡有幾個 1，迴圈就只會精準地跑幾次，不用盲目地把 32 個 bit 全部檢查完。

leetcode 231 Power of Two

class Solution {
public:
    bool isPowerOfTwo(int n) {
        if(n<=0){
            return false;
        }
        else{
            n = n & (n-1);
            if(n==0){
                return true;
            }
            else{
                return false;
            }
        }
        // return n > 0 && (n & (n - 1)) == 0; AI 給的一行解法 
    }
};
